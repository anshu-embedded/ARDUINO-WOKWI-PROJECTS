# 4-Way Traffic Light Controller using Arduino

## Overview

A 4-way traffic light controller simulated using Arduino in Wokwi.

## Features

- Four-way traffic control
- Red, Yellow and Green signal operation
- Timed signal transitions

## Tools Used

- Arduino UNO
- Wokwi Simulator
- Arduino C++

## Traffic Signal States

- Red Light -> Stop
- Yellow Light -> Ready / Wait
- Green Light -> Go
